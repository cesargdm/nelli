# Nelli
> IBM + MNA Watson Cognitive Services app

## Install
Clone the project, install [CocoaPods](https://cocoapods.org) dependencies.

```
pod install
```

## Requirements

Nelli was developed using beta software.
-   Xcode 9.0 beta
-   iOS 11 beta
-   Swift 4
